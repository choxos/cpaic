# Forest plot of relative effects

Forest plot of relative effects

## Usage

``` r
forest(x, ...)
```

## Arguments

- x:

  A `cpaic_effects` data frame from
  [`relative_effects()`](https://choxos.github.io/cpaic/reference/relative_effects.md),
  or a fit object (then
  [`relative_effects()`](https://choxos.github.io/cpaic/reference/relative_effects.md)
  is called on it).

- ...:

  Passed to
  [`relative_effects()`](https://choxos.github.io/cpaic/reference/relative_effects.md)
  when `x` is a fit.

## Value

`x`, invisibly.
